# fwidth

## Description

Texture filter width.
Calculates the sum abs(ddx(<span class='code'><a href="fwidth.html#decl-p" class="code_param">p</a></span>)) + abs(ddy(<span class='code'><a href="fwidth.html#decl-p" class="code_param">p</a></span>)).

# GroupMemoryBarrier

## Description

Group memory barrier. Ensures that all memory accesses in the group are visible to all threads in the group.




## Signature 

<pre>

</pre>

## Availability and Requirements

Defined for the following targets:

#### hlsl
Available in all stages.

#### glsl
Available in all stages.

#### cuda
Available in all stages.

#### metal
Available in all stages.

#### wgsl
Available in all stages.

#### spirv
Available in all stages.

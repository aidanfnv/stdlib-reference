# RayQuery\<rayFlagsGeneric:uint\>\.CommittedRayFrontFace

## Description

Checks if the committed hit is on the front face of a triangle.



## Signature 

<pre>
<span class="code_keyword">bool</span> <a href="../types/rayquery-03/index.html" class="code_type">RayQuery</a>&lt;<a href="../types/rayquery-03/index.html#decl-rayFlagsGeneric" class="code_var">rayFlagsGeneric</a>:<span class="code_keyword">uint</span>&gt;.<a href="committedrayfrontface-09ch.html">CommittedRayFrontFace</a>();

</pre>

## Return value
true if hit is on triangle front face

## Remarks
GLSL/SPIRV only


## Availability and Requirements

Defined for the following targets:

#### glsl
Available in all stages.

#### spirv
Available in all stages.

Requires capability: `spvRayQueryKHR`.
